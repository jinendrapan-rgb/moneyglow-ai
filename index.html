Got a share code from your partner? Paste it here with the shared password to import their data.

⚠️ Importing will MERGE the shared data with yours. No data is overwritten — duplicates are skipped.

❓ How does Family Share work?
1. You and your partner agree on a shared password (e.g., "OurFamily2026")
2. You export your tracker → it's encrypted with that password
3. Share the encrypted code via WhatsApp/email
4. Your partner opens their MoneyGlow AI, pastes the code + same password
5. Their app merges the shared debts/expenses (no duplicates)
6. Repeat anytime to sync changes!

🔒 Privacy: Your data NEVER touches a server. The encryption uses AES-256-GCM. Without the password, the share code is unreadable garbage.
